html中的图像映射

2018-11-14
  • 前端

当我们需要在一张图片上选择一些区域，这些区域能够被用户所点击然后做出一些动作（如跳转到另一个页面）的时候，图象映射便能够很好地完成这项任务。

需要用到的标签

  • img : 定义图像
  • map : 定义图像地图
  • area : 定义图像地图中的可点击区域

创建步骤

  1. 使用 img 标签定义图象

    <img src="./eg_planets.jpg" usemap="#planemap" alt="Planets">

  2. 使用map标签定义图像地图

    <map name="planetmap" id="planetmap">
</map>

    注释：img 元素中的 “usemap” 属性引用 map 元素中的 “id” 或 “name” 属性（根据浏览器），所以同时向 map 元素添加了 “id” 和 “name” 属性。

  3. 使用area标签定义图像地图中的可点击区域

    <area shape="circle"
    coords="180, 139, 14"
    href="" target="_blank" />
<area shape="circle"
    coords="129, 161, 10"
    href="" target="_blank"/>
<area shape="rect"
    coords="0, 0, 110, 260"
    href="" target="_blank"/>

    将这些area标签放到map标签内

整体代码

<img src="eg_planets.jpg" usemap="#planetmap" alt="Planets">
<map name="planetmap" id="planetmap">
    <area shape="circle"
        coords="180, 139, 14"
        href="" target="_blank" />
    <area shape="circle"
        coords="129, 161, 10"
        href="" target="_blank"/>
    <area shape="rect"
        coords="0, 0, 110, 260"
        href="" target="_blank"/>
</map>

效果

点击图片中的星球可在新的标签页打开本页面，这个可根据自己的需求更改
