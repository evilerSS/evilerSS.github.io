预格式化文本

预格式化文本能够将标签内的文本按照自己原来的文本格式显示出来，它保留了空格和换行。

     print i

<pre>

</pre>

“计算机输出”标签

效果

Computer code
Keyboard input
Sample text
Computer variable

代码

<code>Computer code</code>
<kbd>Keyboard input</kbd>
<samp>Sample text</samp>
<var>Computer variable</var>

这些标签常用于显示计算机/编程代码
code 元素不保留多余的空格和折行
kbd 元素定义键盘输入
samp 元素定义计算机输出示例
var 元素定义数学变量
提示：通常，HTML 使用可变的字母尺寸，以及可变的字母间距。在显示计算机代码示例时，并不需要如此。
kbd, samp, 以及 code 元素全都支持固定的字母尺寸和间距。

地址

地址标签为address，使用此标签可以在Html文件中写地址

效果


Written by Eviler.
Visit us at:
eviler.top


代码

<address>Written by <a href="eviler.top">Eviler</a>.
Visit us at:
eviler.top
</address>

删除字效果和插入字效果

效果

一打有 二十 十二 件。

代码

<p>一打有 <del>二十</del> <ins>十二</ins> 件。</p>

大多数浏览器会改写为删除文本和下划线文本。
一些老式的浏览器会把删除文本和下划线文本显示为普通文本。
我们也可以通过改变文本标签的样式来达到这种效果:

<p>一打有 
    <i style="text-decoration: line-through">二十</i>
    <i  style="text-decoration: underline">十二</i> 件。
</p>

或者使用标签s和u：

<p>一打有 <s>二十</s><u>十二</u>件。</p>

块引用

代码

这是长的引用：
<blockquote>
这是长的引用。这是长的引用。这是长的引用。这是长的引用。这是长的引用。

使用 blockquote 元素的话，浏览器会插入换行和外边距，而 q 元素不会有任何特殊的呈现。
